Our mission

We design and maintain services that work with keyboards, screen readers, magnification, and speech input technologies.

Accessibility principles

  • Write plain-language content with semantic structure and meaningful headings.
  • Provide names, roles, and values for all interactive controls.
  • Run automated and manual accessibility checks before each release.
